# README 

## Screen Time App

## Overview
A simple Flutter app that displays your daily, weekly  and monthly screen time using Android's Usage Stats API. It then uses this information to compare against your friends to see who is doing better! The focus is on clarity, minimalism, and taking responsibility.

---

## Installation

### Requirements
- Android device (real device required emulators cannot grant Usage Access)
- Flutter SDK 3.24+
- Android SDK 34+

### Steps
```bash
git clone <Unplug>
flutter pub get
flutter run
```

Once the app is completed, it will be able to be installed through the Google Play store or downloaded to your phone.
